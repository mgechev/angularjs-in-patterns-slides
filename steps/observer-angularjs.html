Observer AngularJS

In JavaScript we have first-class functions, which means that we can pass function instead of observer instances.

In the JavaScript world this pattern is better known as publish/subscribe. 

//Subscribe to event
function ExampleCtrl($scope) {
  $scope.$on('event-name', function handler() {
    //body
  });
}

//Publish event
function ExampleCtrl($scope) {
  $scope.$emit('event-name', { foo: 'bar' });
}
